# pushy-sdk-web
[![npm version](https://badge.fury.io/js/pushy-sdk-web.svg)](https://www.npmjs.com/package/pushy-sdk-web)

The official [Pushy SDK](https://pushy.me/docs/web-push/get-sdk) for Web.

> [Pushy](https://pushy.me/) is the most reliable push notification gateway, perfect for real-time, mission-critical applications.

## Usage

Install the package using npm:

```shell
npm install pushy-sdk-web --save
```

Continue following the instructions on this page:
https://pushy.me/docs/web-push/get-sdk
